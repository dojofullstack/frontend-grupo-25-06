Tipos de etiquetas

3 Tipos de etiquetas :
- block
- inline
- inline-block

Tipo En BLoque (block)

Este tipo de Tag ocupa el 100% del ancho del viewport (puedes controlar sus proporciones ancho y alto) Ejemplos de este tipos:
- div - h1 (titulos) - p (parrafos) - header, section, article

Tipo En linea o (inline)

Solo ocupa el espacio necesario, NO reserva el 100% del ancho: Ejemplos:
- span
- button
- img
- a
- b, mark, i, del (etiquetas de texto enriquecido)
